# Sales Knowledge Hub 

製造業の「営業DX」および「セールス・イネーブルメント」を牽引するAI営業ナレッジベース＆学習ツールのGAS用ソースコードです。

## 初期設定手順

1. **Googleスプレッドシートの作成**
2. 拡張機能 > Apps Script を開き、本リポジトリ内のコードを貼り付けて保存
3. プロジェクト設定の「スクリプトプロパティ」で `GEMINI_API_KEY` を設定
4. エディタから `setupDatabase` 関数を実行しデータベース(シート)を自動構築
5. 右上の「デプロイ」から「新しいデプロイ」(ウェブアプリ)を実行

詳細な手順についてはプロンプト内で説明している通りです。
